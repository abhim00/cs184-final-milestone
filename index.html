Revised Project Proposal

We have decided to change our project to focus on mesh regularization to improve mesh quality for further use. In project 2, we used Edge Flip, split, collapse, and loop subdivision to explore local meshing operators. We hope to extend this project to do mesh regularization of anisotropic meshes.

What We Have Accomplished

In the past week we have familiarized ourselves with Google’s MediaPipe (https://google.github.io/mediapipe/solutions/face_mesh#python-solution-api) and MeshLab (meshlab.sourceforge.net). We have learned how to load images into the software and change the structure of an input mesh. We read about various remeshing algorithms that we can build into our project 2 existing work and decided Hao Li's paper as our “North Star” paper. This work combines the edge flip and split work we did in previous projects with new concepts of curvature and umbilics.

Preliminary Results

The triangle intersection algorithm uses barycentric coordinates to see if our ray intersects a point within a given triangle using the parameter t within the ray. We ensured that the t-value of intersection that we calculated using the discriminant was within the range of the scene. If multiple t values satisfied the terms, we selected the earliest t-value where the ray intersected the object.
